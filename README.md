# Alesmrazek Tap

## How do I install these formulae?

`brew install alesmrazek/tap/<formula>`

Or `brew tap alesmrazek/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "alesmrazek/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
